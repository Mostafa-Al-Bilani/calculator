The C button clears everything and resets the calculator
The D button deletes the last inputted character
The Enter button is used to input numbers of single or multiple digits
The = is disabled when
  • The user has not entred anything yet
  • The user has not pressed enter while inputing a number
  • The calculator is in postfix mode and the last item is a number
  • The calculator is in prefix mode and the last item is an operator
  • The operators' count are not less than the numbers' count by 1
The operators(+, -, x, /) are inputted without using enter
In postfix mode you can't start with an operator you should input at least two numbers first
In prefix mode you can't start with a number you should input an operator first
